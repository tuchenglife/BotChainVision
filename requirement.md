# BotChainVision 需求規格 (Requirements)

> 專案：人形機器人關鍵零組件與台股供應鏈追蹤 Dashboard  
> 最後更新：2026-07-04

---

## 1. 專案目標

建立個人用的 **資料收集 Web Dashboard**，追蹤人形機器人供應鏈相關台股標的之股價、財務與產業動態，並支援手動維護供應鏈廠商清單。

## 3. 供應鏈 Settings（種子資料）

供應鏈廠商存於 `settings/supply_chain_vendors.json`，支援之後手動新增股票代號。

| 關鍵技術/製程 | 對應元件範例 | 代表性台灣供應鏈廠商 |
|-------------|-------------|---------------------|
| 精密傳動 (CNC/培林/螺桿) | 交叉滾柱軸承、關節外殼、小腿結構 | 上銀(2049)、全球傳動(4540)、直得(1597)、大銀(4576)、鈞興(4571) |
| 馬達與驅動系統 (電機) | 無框伺服馬達、電磁煞車 | 台達電(2308)、東元(1504)、大銀(4576)、和大(1536) |
| PCB 電子控制 (電子) | 任務電腦/主控制器、感測器PCB | 台達電(2308)、鴻海(2317)、緯穎(6669)、欣興(3037) |
| CoM/SOM 運算模組 (工業電腦) | 機器人主控運算模組、邊緣 AI、ROS2 控制器 | 研華(2395)、凌華(6166)、研揚(6579)、友通(2397) |
| AI 視覺/感測 (光學/電子) | LiDAR、深度相機、RGB 相機 | 所羅門(2359)、亞光(3019)、先進光(3362)、中光電(5371)、佳能(2374) |
| 機構件與組裝 (CNC/壓鑄) | 關節外殼、機身骨架 | 鴻準(2354)、盟立(2474)、和椿(6215)、高鋒(4510) |

**市場別**：上市 `.TW`、上櫃 `.TWO`（yfinance 代號格式）

## 4. 資料模型

### daily_prices
當日股價、最高、最低、成交量、MA20（20 日均線）

### dividend_yield_daily
系統自動計算：`殖利率 = 近12月股利合計 / 收盤價 × 100`

### dividends
歷史現金股利（除息日、金額）

### historical_eps
季度 / 年度 EPS

### estimates（待實作）
新聞或 yfinance 預估 EPS、營收

### portfolio_holdings（待實作）
券商庫存快照與未實現損益，只顯示券商名稱，不顯示帳號。

### portfolio_realized_trades（待實作）
券商已實現損益，作為資本利得來源。

### portfolio_dividends（待實作）
券商股利所得紀錄，與資本利得分開統計。

### supply_chain_vendors
供應鏈廠商主檔（由 settings 同步）

## 5. 非功能需求

| ID | 需求 |
|----|------|
| NF-01 | 密鑰不得提交至 Git（.env、supabase.txt 已 ignore） |
| NF-02 | 單一 ticker 同步失敗不影響其他標的 |

| NF-04 | 個人免費方案可運行（Supabase Free + GitHub Actions + Streamlit Cloud） |

## 6. 已知限制

1. ~~部分上櫃小票 yfinance 無資料~~ → 已用 symbol_resolver 修復
2. 台股 EPS 覆蓋率因標的而異；yfinance 季度 EPS 常只提供近 3 季，年度 EPS 可補近五年圖表
3. Streamlit Cloud 需使用者自行連結 GitHub 完成首次部署
4. Supabase Free 專案 7 天無活動可能 pause（每日 Actions 寫入可避免）
5. 投資帳戶 MVP 已支援已實現損益與庫存匯入；股利所得待使用者提供股利試算/入帳匯出檔
